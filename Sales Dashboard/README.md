# Sales Dashboard & Forecasting

Contents:

- `SALES DASHBOARD AND FORECASTING.pbix`
- `Sample_Sales_Data.xlsx`

## Open in Power BI

1. Open `SALES DASHBOARD AND FORECASTING.pbix` in Power BI Desktop.
2. If prompted, set the data source to `Sample_Sales_Data.xlsx` in this folder.
3. Refresh to load the latest data.

## Notes

- Swap `Sample_Sales_Data.xlsx` with your dataset and update the source path.
- Forecast visuals may use Power BI's built-in forecasting.
